mt-plugin-publishque-manager
============================

# Publish Queue Manager

## About

公開キューを表示・管理するためのMovable Type プラグインです。

 * リスティングフレームワークで、公開キューなどの Schwarz Job を表示します。
 * 選択した公開キューを削除出来ます。
 * 選択した公開キューを再構築出来ます。
 * 選択した公開キューの優先度を変更出来ます。
